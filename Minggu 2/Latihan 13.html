Bagian 1

PHP

PHP adalah singkatan dari "Hypertext Preprocessor." Ini adalah bahasa pemrograman yang sering digunakan untuk pengembangan aplikasi web. PHP adalah bahasa sumber terbuka (open source) yang sangat populer dan banyak digunakan di seluruh dunia untuk membuat situs web dinamis. PHP digunakan untuk mengolah data yang diterima dari formulir web, mengakses database, menghasilkan halaman web yang dinamis, dan melakukan berbagai tugas lainnya terkait pengembangan web. Kode PHP dapat disisipkan ke dalam dokumen HTML sehingga memungkinkan Anda untuk menciptakan halaman web yang dapat berinteraksi dengan pengguna dan basis data.













Bagian 2

JavaScript

JavaScript (sering disingkat sebagai JS) adalah bahasa pemrograman yang digunakan untuk mengembangkan situs web interaktif dan aplikasi web. JavaScript adalah salah satu dari tiga teknologi inti dalam pengembangan web bersama dengan HTML (Hypertext Markup Language) dan CSS (Cascading Style Sheets). JavaScript berbeda dari Java, meskipun memiliki kata "Java" dalam namanya. Java adalah bahasa pemrograman yang berbeda dan memiliki tujuan yang berbeda. JavaScript, di sisi lain, adalah bahasa skrip yang dijalankan di sisi klien (di browser pengguna) untuk mengendalikan perilaku halaman web secara dinamis. Dengan JavaScript, Anda dapat melakukan berbagai tugas, seperti validasi formulir, animasi, manipulasi DOM (Document Object Model), mengirim dan menerima data dari server tanpa perlu memuat ulang halaman, dan banyak lagi.













Bagian 3

XAMPP

XAMPP adalah perangkat lunak yang digunakan untuk membuat lingkungan pengembangan web lokal yang lengkap di komputer Anda. Nama "XAMPP" adalah singkatan dari "X" yang mengacu pada sistem operasi apa pun (Windows, macOS, Linux), "Apache" sebagai server web, "MySQL" sebagai sistem basis data, "PHP" sebagai bahasa pemrograman server-side, dan "Perl" sebagai bahasa pemrograman tambahan yang juga tersedia. Dengan XAMPP, Anda dapat dengan mudah menginstal dan mengonfigurasi semua komponen ini di komputer lokal Anda, sehingga Anda dapat mengembangkan dan menguji aplikasi web tanpa harus mengunggahnya ke server yang sebenarnya. Ini sangat berguna untuk pengembang web yang ingin mengembangkan dan menguji situs web atau aplikasi web sebelum mereka diterapkan secara online.
